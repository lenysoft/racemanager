# racemanager

the purpose of the application is to manage whole motor race seasons.
you can change tracks, drivers, teams and so on.

## features

- server
- - java ee 7
- - rest api
- client
- - single page application (spa)
- - angular 4

## installation

- requirements
- - maven
- - npm
- - git
- - java 8
- - java ee 7 application server (wildfly 10)

## build

- mvn clean install
